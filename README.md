## Roman to Arabic Converter

Това е Java Swing приложение за конвертиране на числа от римски към арабски и обратно.

## Как да стартираш проекта:
1. Клонирай репозитори-то или изтегли като ZIP.
2. Отвори проекта в предпочитано IDE.
3. Намери файла `src/ConverterApp.java`.
4. Натисни зеления бутон "Run" до `public static void main`.
